Probabilistic Programs [van de Meent et al., 2018]

  • Infer the posterior distribution \(p(\xlat|\xobs)\) over latent variables \(\xlat\) (specified by sample statements) conditioned on observed variables \(\xobs\) (specified by observe statements)
  • We consider the joint distribution over the space of “traces” \((x_{a_t}, a_t)\) for \(t=1,\dots,T\) \[p(\xxx, \aaa) = \prod_{t=1}^T p(a_t|x_{a_1:a_{t-1}},a_{1:t-1}) p(x_{a_t}|x_{a_1:a_{t-1}},a_{1:t})\]
  • We denoted the set of all variables \(\xxx=\xlat\cup\xobs\)
  • Address transitions are deterministic: \(p(a_t|x_{a_1:a_{t-1}},a_{1:t-1})\) is a Kronecker-Delta function
  • But this notation becomes useful later

Inference Compilation (IC) [Le et al., 2017]

  • Self-normalizing sequential importance sampling (SIS)
  • Draw \(K\) samples \(\xlat^k \overset{i.i.d.}{\sim}q(\xlat|\xobs)\) (proposal distribution)
  • Compute the weights, \[w^k = \frac{p(\xlat^k,\xobs)}{q(\xlat^k|\xobs)}\]
  • Approximate the posterior as \[ p(\xxx_{\mr{lat}}|\xxx_{\mr{obs}})\approx\frac{\sum_{k=1}^Kw^k\delta(\xxx_{\mr{lat}}^k-\xxx_{\mr{lat}})}{\sum_{k=1}^Kw^k}\]
  • IC is all about constructing \(q(\xlat|\xobs)\) \(\rightarrow\) efficient SIS \[q(\xxx_{\mr{lat}}|\xxx_{\mr{obs}};\phi) = \prod_{x_{a_t}^{\mr{lat}}\in\xxx_{\mr{lat}}}q(x^{\mr{lat}}_{a_t}|\eta_{a_t}(x^{\mr{lat}}_{<{a_t}},\xxx_{\mr{obs}},\phi))\]
  • \(\eta(\cdot)\) is a recurrent neural network
  • We train \(q(\cdot|\xobs;\phi)\) to be close to \(p(\xlat|\xobs)\) for all \(\xobs\),
    by minimizing the expected KL divergence under the marginal \(p(\xxx_{\mr{obs}})\) \[\begin{align}\mL_{\mr{IC}}(\phi)&=\meanp{p(\xxx_{\mr{obs}})}{\kl{p(\xxx_{\mr{lat}}|\xxx_{\mr{obs}})}{q(\xxx_{\mr{lat}}|\xxx_{\mr{obs}};\phi)}}\nn\\ &=-\meanp{p(\xxx)}{\log q(\xxx_{\mr{lat}}|\xxx_{\mr{obs}};\phi)} + \mr{const}\nn\end{align}\]
  • “Evaluation based inference” requires the simulator to run repeatedly

Probabilistic Surrogate Network (PSN)

  • IC is still limited by the execution speed of the program
  • How about learning a surrogate model \(s\) that replaces \(p\) as a program when \(p\) is slow?
  • We structure \(s\) as a distribution over traces factorized exactly like \(p\) \[ s(\xxx,\aaa;\theta)=\prod_{t=1}^{T}s(x_{a_t}|\xi_{a_t}(x_{a_1:a_{t-1}},a_{1:t},\theta))s(a_t|\xi_{t-1}(x_{a_1:a_{t-1}},a_{1:t-1},\theta))\]
  • \(\xi_{a_t}(\cdot)\) and \(\xi_t(\cdot)\) are recurrent neural networks
  • We match \(p\) and \(s\) by minimizing the KL-divergence \[\begin{align*} \mL(\theta) &= \kl{p(\xxx,\aaa)}{s(\xxx,\aaa;\theta)} \\ &= -\meanp{p(\xxx,\aaa)}{\log s(\xxx,\aaa;\theta)} + \mr{const} \end{align*}\]
  • Now we have to model the address transitions \(s(a_t|\xi_{t-1}(x_{a_1:a_{t-1}},a_{1:t-1},\theta))\)
  • For instance a categorical distribution
  • Fully compatible with the probabilistic programming framework
